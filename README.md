# Swift App Architecture

A Claude Code plugin providing architectural guidance for Swift application development.

## Installation

```bash
claude mcp add-plugin gestrich-swift-app-architecture
```

Skills are invoked via slash commands in Claude Code (e.g., `/skill-name`).
